What will we learn in this course?

In the first week, we will establish a solid foundation in Java programming language.

  1. Java language syntax
  2. Basic programming principles in Java
  3. How to test your code

During the second week, we will look at some standard Java libraries to boost our productivity.

  1. Some of Java's standard libraries
  2. Common design patterns and in Java
  3. You will understand the SOLID principle

During the third week, we learn to how to write a Java application backend with REST API

  1. MVC using Spring
  2. You will implement your own application backend with REST API
