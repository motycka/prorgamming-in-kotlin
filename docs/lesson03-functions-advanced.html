The Lambda Function

Anonymous functions, or lambda functions, are functions that are not declared, but passed immediately as an expression.

A lambda function in Kotlin is an anonymous function that can be treated as a value. It can be passed as an argument to other functions, returned from other functions, or assigned to variables.

The basic syntax of a lambda function is: 

						
							val lambdaName: (Type) -> ReturnType = { argument: Type -> body }

Here, a lambda function is assigned to a variable greet that takes an argument of type String and prints a greeting message. 

						
							val greet: (String) -> Unit = { name: String -> println("Hello, $name!") }

							greet("World") // Output: Hello, World!

Common examples of lambda function is the forEach: 

						
							val cities = listOf("Bangkok", "Barcelona", "Tokyo", "London", "New York")

							cities.forEach {
								println(it)
							}

The Lambda Function

The main use of lambda functions is to pass them as arguments to higher-order functions. 

						
							fun operation(x: Int, y: Int, func: (Int, Int) -> Int): Int {
								return func(x, y)
							}

							val sum = operation(10, 20) { x, y -> x + y }

Scope Functions

Scope functions are functions that allow you to execute a block of code within the context of an object.

When you use a scope function, you can access the object's properties and functions without having to use the object's name.

The scope functions in Kotlin are let, run, with, apply, and also.

let

The let function allows you to execute a block of code on a result of call chains (object).

It is often used to execute a block of code on the result of a call chain, such as filter, map, or any other operation that returns a value. 

						
							val cities = listOf("Bangkok", "Barcelona", "Tokyo", "London", "New York")

							val result = cities
								.filter { it.startsWith("B") }
								.let { println(it) }

The let function is particularly useful when used with a nullable object. 

						
							val name: String? = "John"

							name?.let { println("Name is $it) } ?: println("Name is was not given")
